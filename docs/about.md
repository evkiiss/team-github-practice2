# О проекте
Этот проект создан для тренировки совместной разработки через GitHub.
## Что мы изучаем
- git clone;
- git add;
- git commit;
- git push;
- git pull;
- ветки;
- pull request;
- merge conflict.
## Почему важно работать в команде
Совместная разработка помогает распределять задачи, быстрее находить ошибки и
учиться читать чужой код.
